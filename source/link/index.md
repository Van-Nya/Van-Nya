## 如何添加自己的链接

1. 在评论区发送如下代码块  
（评论区支持 Markdown）

    ```YAML
    - name: <网名>
      link: <主页链接>
      avatar: <头像链接>
      descr: <描述>
    ```

1. 在自己的网站将 [**Van喵的小窝**](https://van.ここ.com) 添加为友情链接（可选

    ```YAML
    - name: Van_Nya
      link: https://van.ここ.com
      avatar: https://ruavan.codes/img/avatar.png
      descr: 一只想贴贴的小猫（确信
    ```
